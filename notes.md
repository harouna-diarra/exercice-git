Git et GitHub sont des outils utilisés pour le travail colaboratif;
Git permet de faire des commit, pull ...
GitHub est accesible vi site WEb
le principe est simple et suit ces étapes:
git add pour mettre dans la zone de préparation;
git commit pour mettre dans le dépôt local;
git push pour mettre dans le dépôt distant;
git pull pour prendre du code dans le dépôt distant;
